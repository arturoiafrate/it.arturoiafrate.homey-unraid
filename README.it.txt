Integrazione non ufficiale per Unraid.
Questa integrazione permette di monitorare e gestire il tuo server NAS Unraid.

Lingue supportate:
English
Italiano

Come funziona?
Installa l'app e aggiungi un nuovo dispositivo unraid. Nelle impostazioni configura:
• Host > l'hostname/indirizzo ip del server Unraid.
• Username > un utente valido  del Server Unraid. Puoi utilizzare anche l'utente root.
• Password > la password dell'utente
• Port > la porta utilizzata per la connessione  SSH. Default: 22
• Ping interval > intervallo di ping (in secondi). Utilizzato per verificare se il server Unraid è online.
• MAC address > l'indirizzo MAC del server Unraid. Se valorizzato e se il server Unraid si trova nella stessa LAN del tuo Homey, puoi inviarie un WOL per risvegliare il NAS.
• Check interval > intervallo di refresh (in secondi) dei sensori.

Per maggiori informazioni sull'installazione del servizio visita il repository github.
